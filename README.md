## 📁 How to Run Locally

1. Clone the repo:

```bash
git clone https://github.com/yuvraj-3/Resume.git
cd Resume
```
Install requirements:

```bash
pip install -r requirements.txt
```
Run the app:

```bash
streamlit run app.py
```
